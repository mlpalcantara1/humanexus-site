# HUMANEXUS Site

Aplicação institucional isolada do sistema principal HUMANEXUS.

## Requisitos

- Node.js 20+
- npm 10+

## Executar

```bash
npm install
npm run dev
```

## Produção

```bash
npm run build
npm run start
```

## Observação

Esta aplicação não possui integração real com banco, APIs, autenticação ou dashboards do sistema HUMANEXUS principal.
